Tribo Interactive
=================

Site desenvolvido em **ASP.NET MVC 4**

## Dependências

Para o desenvolvimento você precisa ter `Git` + `Node.js` + `NPM` + `Ruby`.  
Lei o arquivo `SETUP.md` para saber como instalá-los.

## Desenvolvimento

    grunt watch

Geram os arquivos `output.js` e o `screen.css`.

## Estrutura do Site

    - Home
    - A Tribo
    - O que fazemos
    - Trabalhos
        |- Detalhe do projeto
    - Contatos
    - Últimas
        |- Notícia comleta
    - Busca
